# Objetivos del proyecto

## Objetivo general

Proveer un punto de contacto centralizado que funcione como la principal
interfaz digital de comunicación y conversión del Arequipa Data Fest 2026,
permitiendo transmitir la identidad del evento y facilitar el acceso a
información relevante para los participantes.


## Objetivos específicos

1. Transmitir la identidad visual y propuesta de valor del evento mediante
una experiencia visual atractiva.


2. Presentar información crítica del evento:
- Fecha.
- Hora.
- Ubicación.
- Agenda.
- Ponentes.


3. Facilitar el registro de participantes mediante integración con la
plataforma externa Luma.


4. Garantizar una experiencia responsive en dispositivos móviles y
escritorio.


5. Implementar una arquitectura frontend mantenible y escalable.
