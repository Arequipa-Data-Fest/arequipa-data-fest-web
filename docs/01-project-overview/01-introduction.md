# Introducción

## Nombre del proyecto

Arequipa Data Fest 2026 Website


## Descripción

El proyecto consiste en el desarrollo del sitio web oficial del
Arequipa Data Fest 2026, un evento especializado en datos que busca
conectar profesionales, estudiantes y entusiastas del ecosistema de
datos.

El sitio web funcionará como el principal punto de comunicación,
información y conversión del evento.


## Propósito

Proporcionar una experiencia digital clara y atractiva que permita a
los visitantes conocer el evento, explorar información relevante y
acceder al proceso oficial de registro.


## Contexto

El evento requiere una plataforma web capaz de soportar visitas
simultáneas durante la etapa de difusión y lanzamiento.

Debido a que el registro será gestionado externamente mediante Luma,
el sistema se enfocará principalmente en presentación de información,
experiencia de usuario y conversión.
